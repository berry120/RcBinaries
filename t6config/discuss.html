wayshwing:
I had seen a LOT of forums online, complaining about the connectibility or lack thereof, of the USB programming cable for the FS-CT6A radio. It seems that the PL-2303 USB chip used in the cable is not Windows friendly, and basically connect to Windows by sheer luck  >:(

After a LOT of researching and trial and error, I fould a $4 solution!!!  ;D ;D ;D

On XHeli.com, they sell whats called a "computer simulator cable" in their flight sim section. It is a 4-pin to serial cable, which connects your radio to the PC using a serial port. NO USB DRIVER NEEDED! I use it and it works great.  ;D

As a word of advice, if you want to use a USB connection (if your PC has no serial ports), uses this $4 serial cable with a separate USB-to-serial adapter. Avoid using a USB adapter which uses the "PL-2303" USB adapter chip. This cheap Taiwanese USB device chip is very unstable, and rarely connects with Windows due to crazy weak compatibility drivers.  >:(  Find a USB-to-serial adapter that uses a different chip.

dditch66:
Here is the info that some have used to getting the prolithic driver working which I had previously posted on the blueray support thread:

If you have the RTF version here is the pocedure for using the t6config.exe program and driver:

  Install the USB driver first. (PL-2303 Driver Install.exe)
  Install the t6config.exe program next.
  Plug the USB cable into the Computer.
  Click through the hardware detection screens.
  After the USB hardware detection screens, turn on the power to the Tx (did you install batteries?)
  run T6config.exe
  Click the "Setting" button to select the serial port that was created by the USB install (likely the last on the list).

  If it is communicating, you'll see that moving your sticks make the green/black bars at the top move.
  Click "Get User" to get the Tx setup loaded into your PC.
  Click "Save" to save your settings (do this before you make changes so you have your defaults saved).

  Note: Do not edit or save your config files without the Tx plugged in and communicating. I have found at least on my system, if I do this, it does not always save the data right if you work w/ T6config.exe when the transmitter is not plugged in and communicating w/ the PC.

I did not have any issues w/ installing my USB driver.
I installed the driver BEFORE plugging in the USB device.
But if you do, I heard this helps.
Download the MS Windows serial USB system file, called "ser2pl.sys" from here:

http://www.floridaprobe.com/usb_install.htm

The cable uses the Prolific PL-2303 USB chip. Apparently it doesn't always get recognized by Windows, so the system file doesn't get installed correctly.

Download the system file, and copy the file to your C:\Windows\System32 folder.
Plug in the USB cable into your computer, then restart your computer with it plugged in. It "should" get recognized at startup.
